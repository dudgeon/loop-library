# Goal — root (meta) loop

> **An enduring primitive** (hypotheses [H1](HYPOTHESES.md) / [H2](HYPOTHESES.md)): a loop has a
> persistent goal and a defined work product. This file instantiates them for the repo's root
> loop. Alpha — revise freely.

## Goal (enduring)
Build a great loop library:
- **Purpose 2 — research** how loops are constructed and recursively improved;
- **Purpose 1 — ship** validated, vendorable loop kits.

Minimal phrasing: *make a great, compounding knowledge base about loops — and ship the best of
it.* The goal persists across iterations; it is not a task that completes.

## Work product
The repo itself — **self-referential**: the `wiki/` corpus (the research loop) plus, eventually,
the kits in `dist/` (shipped loops). The work product of the root loop is *a better root loop and
better shipped loops.*

## Stop condition
**None.** The root loop is enduring (steady-state improvement), not a delivery loop that
terminates. Termination is a property of *tasks* ([`TASKS.md`](TASKS.md)) and of shipped `dist/`
kits — not of the meta loop.
